=============
Odoo Sentinel
=============

This is an Odoo specific ncurses client for the stock_scanner module.

It allows to run the scenarios on any text-based terminal.

Command line arguments
----------------------

- -p: OdooRPC profile to use for Odoo connection.
- -c: OdooRPC's configuration file to use.
- -l: Logfile where to write logs.
- -t: Test file to read for inputs.
- -b: Audio file for a different Beep sound
